# Overview
- Operating System: **Arch Linux**
- Window Manager: **Sway**
- Login Manager: **ly**
- Terminal: **Kitty**
- Shell: **Zsh**
- Status Bar: **Waybar**
- App Launcher: **Rofi-Wayland**
- Sound Menu: **Pavucontrol**
- File Manager: **Yazi**
- Screen Lock: **Swaylock**
- Logout Menu: **Wlogout**
